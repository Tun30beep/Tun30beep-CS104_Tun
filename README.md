# Restaurant Management System

Restaurant Management System developed using Flask and SQLite.

---

## Features

- Manage Menu
- Add / Edit / Delete Menu
- Manage Customers
- Manage Employees
- Manage Restaurant Tables
- Add Orders
- Dashboard System
- SQLite Database Integration

---

## Technologies Used

- Python
- Flask
- SQLite
- HTML
- CSS
- GitHub
- PythonAnywhere
